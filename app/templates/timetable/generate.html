About the Hybrid GA+CSP Algorithm
1
CSP Initialization

Constraint Satisfaction Problem solver creates a valid initial solution using backtracking with MRV (Minimum Remaining Values) heuristic.

2
Genetic Algorithm Optimization

Evolution-based optimization improves the solution through selection, crossover, and mutation over multiple generations.

3
Constraint Checking

Hard constraints (no clashes) are enforced strictly. Soft constraints (preferences, distribution) are optimized for fitness score.
